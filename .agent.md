---
# Agent customization frontmatter
# `applyTo` はこの設定を適用するファイルパターン（glob）です。
applyTo:
  - "**/*"

# 外部指示ファイルを明示して Agent が自動的に読み込めるようにする
instructionsFile: "copilot-instructions.md"

---

説明
- このファイルはローカル Agent に対するカスタム設定例です。ワークスペース内の Agent/ツールが YAML frontmatter を参照する実装であれば、`instructionsFile` に指定した `copilot-instructions.md` を自動で読み込みます。

注意
- 実際に自動読み込みするためには、使用中の Agent 実装（VS Code 拡張や CI ツール）が `.agent.md` の `instructionsFile` を解釈する必要があります。解釈しない場合は、プロンプト実行時に `copilot-instructions.md` の要点を明示的に渡してください。

カスタマイズ例
- 特定フォルダだけに適用する場合は `applyTo` を調整してください:

  applyTo:
    - "src/**"
